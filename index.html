Guía / Glosario
Guía y Glosario

Glosario

Switch: Dispositivo que conecta equipos dentro de una misma red (LAN) usando direcciones MAC.
Router: Dispositivo que conecta diferentes redes y dirige el tráfico entre ellas usando direcciones IP.
Gateway: La dirección IP del Router que permite a un equipo salir a otras redes (Internet).

Cómo usar

  • Arrastra dispositivos desde la barra lateral al área de trabajo.
  • Usa la herramienta Conectar para crear cables entre interfaces.
  • Configura IPs haciendo clic en cada dispositivo (Puntero).
